## Description

Orthodox Companion helps Eastern Orthodox Christians follow the daily rhythm of the Church with fasting guidance, liturgical readings, feast countdowns, study tools, icon and chant libraries, prayer requests, church finder tools, language learning, and a source-backed Orthodox assistant prototype.

The app includes personalization, simple mode for easier reading, multilingual settings, and an iOS-inspired design for calm daily use.

## Review Notes

Orthodox Companion can be used without signing in. Account sign-in is optional and uses Firebase Authentication. Location is optional and requested only when the user chooses Find Near Me in the Church Finder.

Admin tools are hidden from normal users and are intended for future editorial workflows. They are not required for normal app review.

If the reviewer asks for sign-in testing, create a temporary Firebase Auth test user in Firebase Console and provide the email/password in App Store Connect review notes.
