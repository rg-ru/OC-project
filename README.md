# OC-project
Build a production-ready mobile app called "Orthodox Companion" for iOS, Android, and Web using Flutter and Firebase.

Mission:
Create the most complete Orthodox Christian app available, focused on Eastern Orthodoxy. The design should be beautiful, modern, reverent, and easy to use. Use a dark mode inspired by Byzantine iconography with gold accents.

Core Features:

1. Orthodox Fasting Calendar

* Daily fasting rules.
* Show allowed foods.
* Support multiple jurisdictions.
* Liturgical calendar integration.
* Feast and fast periods highlighted.
* Calendar view and daily view.

2. Feast Day Countdown

* Countdown to major feasts.
* Push notifications.
* Feast descriptions.
* Icons and historical information.

3. Orthodox Quote Generator

* Quotes from St. Paisios, St. Porphyrios, St. Seraphim of Sarov, St. John Chrysostom, St. Ignatius Brianchaninov, St. Theophan the Recluse, and other Orthodox saints.
* Daily quote widget.
* Save favorites.
* Share quotes.

4. Church Finder

* Interactive map.
* Search nearby Orthodox churches.
* Filter by jurisdiction.
* Directions integration.
* Service schedules.
* Church contact information.

5. Orthodox Study Center

* Interactive quizzes.
* Catechism lessons.
* Church history lessons.
* Scripture studies.
* Progress tracking.
* Achievement system.

6. Daily Gospel Readings

* Daily Epistle and Gospel.
* Liturgical readings.
* Audio reading option.
* Offline reading cache.
* Bookmark verses.

7. Orthodox Icon Encyclopedia

* Searchable icon database.
* High-quality icon images.
* Saint biographies.
* Feast information.
* Historical explanations.
* Save favorites.

8. Bible Reading Tracker

* Reading plans.
* Progress statistics.
* Daily reminders.
* Multiple Bible plans.
* Notes and bookmarks.

9. Prayer Requests

* Community prayer wall.
* Anonymous requests.
* Prayer counters.
* Moderation tools.
* Prayer notifications.

10. Orthodox Language Learning

* Church Slavonic basics.
* Liturgical Greek basics.
* Vocabulary lessons.
* Flashcards.
* Pronunciation audio.
* Progress tracking.

11. AI Orthodox Assistant

* AI-powered Q&A assistant.
* Only answer using approved Orthodox sources.
* Cite sources.
* Explain theology simply.
* Support multiple languages.
* Conversation history.

12. Sermon Library

* Search sermons.
* Filter by saint, topic, feast, or scripture.
* Audio playback.
* Transcript support.
* Bookmark sermons.

13. Liturgical Calendar API

* Build a public REST API.
* Daily readings endpoint.
* Saints endpoint.
* Feast endpoint.
* Fasting endpoint.
* Authentication and rate limiting.

14. Orthodox Chant Library

* Byzantine chants.
* Russian chants.
* Serbian chants.
* Greek chants.
* Audio player.
* Playlist support.
* Offline downloads.

15. Interactive Church History Timeline

* Timeline from Pentecost to present.
* Ecumenical councils.
* Saints.
* Major events.
* Interactive maps.
* Visual learning experience.
